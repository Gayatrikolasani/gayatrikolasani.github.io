AI-Powered Career Pathway Guidance

Intelligent recommendation system for personalized career paths and skill-gap insights.

Overview

This project focuses on helping students and graduates discover suitable career paths by combining their profile data with real-time job market trends. The system recommends roles, highlights skill gaps, and suggests relevant learning resources.

What I did

  • Built a full end-to-end ML pipeline using Python and Scikit-learn.
  • Applied TF-IDF (and experimented with Word2Vec) to convert text inputs into numerical vectors and reduce noise.
  • Used K-Nearest Neighbors and XGBoost to rank and recommend career pathways.
  • Performed skill-gap analysis and mapped users to courses, tools, and certifications to improve employability.
  • Integrated a feedback loop so user responses are used to refine future recommendations.
  • Evaluated alignment between user expectations and predictions using Cohens Kappa.

Impact

  • Generated 500+ personalized career recommendations.
  • Improved feature extraction speed and model performance.
  • Delivered high user satisfaction and more relevant results.

Autonomous Shopping Trolley

Robotics prototype for smart transport in retail and industrial spaces.

Overview

Built during a 36-hour national-level hackathon, this autonomous trolley is designed to follow or navigate defined paths to carry goods across retail stores, warehouses, or healthcare facilities.

What I did

  • Used ROS and Python to control movement, sensing, and navigation.
  • Designed the overall system behaviour and tuned parameters to reduce execution errors by around 60%.
  • Led a team of six, coordinating roles, debugging, and testing in a tight 36-hour timeframe.
  • Created 3D visualization & simulation to improve situational awareness and path validation.
